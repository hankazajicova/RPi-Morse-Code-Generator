## RPi-Morse-Code-Generator

- Morse code translator for Raspberry Pi

- Run with:
    ```bash
    ./run.py text you want to generate to morse code
    ```

# Doveloped for:
- Raspberry Pi 1 with GPIO led diod
- led diod switched in 17 GPIO pin
